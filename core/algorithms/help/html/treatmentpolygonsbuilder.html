This algorithm was designed to create the polygons treatment using GPS points collected in the field.

• GPS points layer: Select the layer with the GPS points. • Reproject if CRS is geographic: Check if GPS layer are in a geographic system. • Suggested CRS: Suggested projected CRS. • Sorting variable: Select the field with the GPS points ID. • Method: Choose the method of polygon construction. • Border size: Specify the size to cut off the area border.
