This algorithm was designed to create a custom presentation based in a template with the analysis results.

• Trial name: Select the name from a list of available trials. • T1 surface points layer: Select the shapefile exported from the T1 final surface raster. • T2 surface points layer: Select the shapefile exported from the T2 final surface raster. • Yield field: Choose the field in which the values are being evaluated. • T1 validation points layer: Select the shapefile with 20% of the features from T1. • T2 validation points layer: Select the shapefile with 20% of the features from T2. • Gain points layer: Select the shapefile exported from the gain surface raster. • Output folder: Specify the folder where the presentation report will be saved.
