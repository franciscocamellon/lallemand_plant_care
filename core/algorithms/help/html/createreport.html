This algorithm was designed to create a custom report .docx file based in a template with the analysis results.

• Trial name: Select the name from a list of available trials. • Yield layer (T1 and T2): Select the shapefile resulted in the filtering process, with T1 and T2 features. • T1 total points layer: Select the shapefile with all features from T1. • T2 total points layer: Select the shapefile with all features from T2. • T1 surface points layer: Select the shapefile exported from the T1 final surface raster. • T2 surface points layer: Select the shapefile exported from the T2 final surface raster. • Gain points layer: Select the shapefile exported from the gain surface raster. • Output folder: Specify the folder where the presentation report will be saved.
