This algorithm was designed to divide the points from the harvester into T1 and T2 layers.

• Yield layer (T1 and T2): Select the shapefile resulted in the filtering process, with T1 and T2 features. • Field to evaluate: Choose the folder containing images related to the point data. Select the folder where your images are stored. • Treatment names: Select the treatments to participate in filtering process. • T1 layer output folder: Specify the folder where the T1 processed output will be saved. • T2 layer output folder: Specify the folder where the T2 processed output will be saved.
