This algorithm was designed to create two samples from a given layer.

• Treatment points layer: Select the shapefile with all features from T1 or T2. • Sample percentage: Specify the percentage for the main sample. • Complementary percentage: Specify the percentage for the complementary sample. Must sum 100% with the main sample percentage. • Treatment sample output layer: Select the path for the output main sample layer. • Treatment Complementary output layer: Select the path for the output complementary sample layer.
