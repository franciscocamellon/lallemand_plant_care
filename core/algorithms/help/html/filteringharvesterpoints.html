This algorithm was designed to run an R script developed by the LPC Team. It filters the harvester points layer by custom parameters.

• Harvester points layer: Select the shapefile exported from the harvester. • ID field: Specify the id field that identifies each harvester point. • Yield field: Specify the field which has values to evaluate. • Treatment polygons layer: Select the shapefile with the treatments polygons. • Treatment identifier field: Specify the id field that identifies each treatment polygon. • Contour polygon layer: Select the shapefile with the boundary polygon. • Target projection: Specify the target projection. • Date column: Specify the date column.
