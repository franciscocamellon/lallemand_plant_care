This algorithm was designed to calculate an error compensation.

• T1 80% raster: Select the raster created from the kriging of 80% of the T1 points. • T1 error raster: Select the raster created from the kriging of 20% of the T1 points. • T2 80% raster: Select the raster created from the kriging of 80% of the T2 points. • T2 error raster: Select the raster created from the kriging of 20% of the T2 points • Export final surface points: Select to generate a point layer from the compensation.
