This algorithm was designed to create the sample layers which one with 80% and 20% of the features respectively. After that this algorithm exports a histogram from a chosen attribute.

• Yield filtered layer: Select the shapefile resulted in the filtering process, with T1 and T2 features. • Field with treatment information: Specify the field in which has the information that specify the feature's treatment. • Yield field to histogram: Specify the field in which has the value to compute the histogram.
